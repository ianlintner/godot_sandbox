# godot_sandbox

A 2D platformer game project built with Godot Engine, inspired by classic NES Capcom games like Duck Tales and Mega Man.

## Overview

This repository contains a complete MVP (Minimum Viable Product) for a retro-style 2D platformer game featuring:
- Player character with movement, jumping, and shooting
- Enemy AI with patrol behavior
- Health/life system
- Platform-based level design
- Hit detection and collision systems

## Quick Start

1. Install [Godot Engine 4.2+](https://godotengine.org/download)
2. Clone this repository
3. Open the project in Godot
4. Press F5 to run the game

## Documentation

For detailed game information, controls, and technical details, see [GAME_README.md](GAME_README.md).

## Project Structure

- `scenes/` - Godot scene files (.tscn)
- `scripts/` - GDScript files (.gd)
- `assets/` - Game assets (sprites, sounds, etc.)
- `project.godot` - Main project configuration
